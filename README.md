# React Demo App

A React application, built with Vite and styled with Tailwind CSS. A secure login flow, dynamic live-filtered list searching, detailed item views, and a unit testing suite using Vitest and React Testing Library.

🔗 Live Demo: https://react-demo-app-2026.vercel.app/

## 🚀 Local Setup & Installation

### 1. Clone the Repository

### 2. `cd` into `react-demo-app`

### 3. Install Dependencies using `npm install`

### 4. Run the Development Server using `npm run dev`

### 5. Run Tests in Terminal using `npm run test`

### 6. Run Tests in Interactive Web GUI using `npm run test:ui`
